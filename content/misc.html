Recent R package releases

Last update: 2022-01-12

We regularly update recently released R packages handling missing values. This is a non-exhaustive list, if you have suggestions feel free to contact us.

  • PKLMtest: Implementation of a KL-based (Kullback-Leibler) test for MCAR (Authors: Meta-Lina Spohn, Loris Michel, Jeffrey Naef; related article)

  • Iscores: Proper Scoring Rules for Missing Value Imputation (Authors: Loris Michel, Meta-Lina Spohn, Jeffrey Naef; related article)

  • NADIA: NA Data Imputation Algorithms (Authors: Jan Borowski, Piotr Fic, Katarzyna Woznica)

  • misaem: Logistic regression with missing covariates—Parameter estimation, model selection and prediction within a joint-modeling framework (Authors: Jiang W, Josse J, Lavielle M, Group T)

  • imputeFin: Imputation of Financial Time Series with Missing Values (Authors: Junyan Liu, Daniel P. Palomar)

  • missSBM: Handling Missing Data in Stochastic Block Models (Authors: Julien Chiquet, Pierre Barbillon, Timothée Tabouy)

  • RMixtComp: Mixture Models with Heterogeneous and (Partially) Missing Data (Authors: Vincent Kubicki, Christophe Biernacki, Quentin Grimonprez)

  • ABSLOPE: High-dimensional Model Selection with Missing Values (Author: Wei Jiang)

  • bnstruct: Bayesian Network Structure Learning from Data with Missing Values (Authors: Francesco Sambo, Alberto Franzin)

  • psfmi: Predictor Selection Functions for Logistic and Cox regression models in multiply imputed datasets (Author: Martijn Heymans)
